Oscanner

Oscanner is an attendance tracking system using QR codes, with data visualization tools. It prioritizes security with Google Auth and Recaptcha V3, and its UI is built with TailwindCSS. Powered by Laravel and Laravel Breeze, it offers QR code generation and scheduling functionalities for efficient attendance management.

Tech Stack: PHP, Laravel, D3.js, Laravel Breeze, HTML, CSS, JS, TailwindCSS, Google Auth, Recaptcha V3, Endroid QR Code, Flatpickr, Html5 QR Code, PHPSpreadSheet, DomPdf, TUI Calendar, Nominatim.
